วันก่อนเจมส์ไปเจอบทความมา เห็นแล้วแบบ เห้ย สุดยอดดด ทำแบบนี้ได้ด้วยหรอ คือ แต่งหน้า Profile ใน GitHub (บทความที่เจอมาคือบทความที่อยู่ใน Reference อันแรกครับ) และเจมส์ก็ไปหาข้อมูลเพิ่มเติมมา ถ้าหากเราไม่มีหัวในการออกแบบ ก็ Template ต่าง ๆ ให้เราดูเป็ตัวอย่าง และปรับมาเป็นในรูปแบบของเรา

มาเริ่มกันเถอะ

การทำจะเปลี่ยนหน้า Profile ใน Github ของเราได้ คือเราต้องสร้าง Project ที่เป็น public โดยชื่อโปรเจคเป็นชื่อเดียวกับ username ขึ้นมาก่อนครับ เช่น เวลาเข้า github ของเจมส์ เป็นลิงค์ https://github.com/jame3032002 เวลาสร้าง Project เจมส์ก็สร้าง ชื่อ jame3032002 และตั้งเป็น public ด้วยเน้อครับ

เมื่อสร้าง Project ขึ้นมาเรียบร้อยแล้ว ถ้าหากคุณผู้อ่านไม่ได้ติ๊กในส่วนให้มันสร้าง README file มาให้ มันจะแสดงคล้าย ๆ แบบนี้ครับ

แต่ถ้าหากคุณผู้อ่านเลือกสร้าง README file ก็สามารถแก้ไขไฟล์ README.md ได้เลยครับ ในส่วนนี้จะเป็น Profile

ถ้าหากเพื่อน ๆ ยังไม่ทราบว่าจะสร้างยังไง เรามีเว็บที่ Generate ไฟล์ README.md มาให้ครับ https://rahuldkjain.github.io/gh-profile-readme-generator

เมื่อเรา Generate ไฟล์เรียบร้อย เราก็อาจจะมีปรับแต่งให้เป็น Style เราอีกรอบ หรือเราจะดู Template ต่าง ๆ จากลิงค์นี้ก็ได้ครับ มีคนทำไว้เยอะแยะเลยครับ

เรามาว่ากันต่อกับกรณีที่ไม่ได้สร้าง README.md ละกันเน้อครับ ให้เราเปิด Terminal แล้วก็ Clone Project ที่เราสร้างเมื่อกี้ลงมาครับ

เมื่อเรา clone project ลงมาเรียบร้อยแล้วให้เพิ่มไฟล์ README.md แล้วก็ใช้ไอเดีย ตกแต่งได้เลยครับ หรืออาจจะ generate profile จากลิงค์ที่ใส่ไว้ข้างต้นช่วยสร้างให้แล้วมาปรับได้เลยครับ

เมื่อตกแต่งเรียบร้อยแล้วก็ push ขึ้นได้เลยครับ ถ้าหากเพื่อน ๆ อยากดูของเจมส์ก็สามารถเข้าใน profile เจมส์ได้เลยครับ
